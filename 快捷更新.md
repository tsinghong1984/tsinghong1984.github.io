# 快捷更新指南

## 写新文章

在 `_posts/` 下创建文件，命名：`YYYY-MM-DD-标题.md`

模板：

```markdown
---
layout: post
title: "文章标题"
date: 2026-06-15
tags: [网络技术]
---

正文用 Markdown 写...
```

## 标签

在 front matter 里手写 `tags`，想加几个加几个，完全灵活：

```yaml
tags: [网络技术]
tags: [里德堡原子, RAOFDM]
tags: [AI大模型, 论文笔记]
```

## 可用标签（推荐但不限制）

`无线通信` `里德堡原子` `网络技术` `AI大模型`

## 推送到 GitHub

```bash
# 切换到项目目录
cd D:\yuproject\graduate_life\AI\git\tsinghong\tsinghong1984.github.io

# 三步推送
git add -A
git commit -m "描述改动"
git push origin main
```

## 验证

1. 打开 `https://github.com/tsinghong1984/tsinghong1984.github.io/actions` 看构建是否 ✅ 通过
2. 构建通过后访问 `https://tsinghong1984.github.io`
3. 如果页面没变，按 `Ctrl + Shift + R` 强制刷新
