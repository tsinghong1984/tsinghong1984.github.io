# 快捷更新指南

## 写新文章

在 `_posts/` 下创建文件，命名：`YYYY-MM-DD-标题.md`

模板：

```markdown
---
layout: post
title: "文章标题"
date: 2026-06-15
---

正文用 Markdown 写...
```

## 文件夹自动标签

**把文章放到 `_posts/` 的子文件夹中，标签自动生成：**

```
_posts/
├── 网络技术/     ← 文章放这里 → tags: [网络技术]
├── 无线通信/     ← 文章放这里 → tags: [无线通信]
├── 里德堡原子/   ← 文章放这里 → tags: [里德堡原子]
└── AI大模型/     ← 文章放这里 → tags: [AI大模型]
```

- 想新建标签？直接创建新文件夹，然后在 `_config.yml` 的 `defaults` 中加一行映射
- 如果需要一篇文章多个标签，在 front matter 中手动加 `tags: [额外标签]`
